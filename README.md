# 🍽️ FoodLink - Smart Food Waste Tracker for Communities

FoodLink is an AI-powered food redistribution platform developed during **FSM Hackathon 4.0**. It aims to reduce food wastage by connecting food donors with NGOs through intelligent priority-based matching.

## 🚀 Features

- 🤖 AI-powered donor-NGO matching
- 📍 Priority allocation based on expiry time, distance, and quantity
- 🥗 AI Recipe Assistant
- 🌱 AI Composting Assistant
- 📊 ESG Impact Dashboard
- ✅ FSSAI-compliant food verification
- 📈 Real-time food waste analytics

## 💡 Problem Statement

Millions of tons of edible food are wasted every year while many people struggle with food insecurity. FoodLink provides an intelligent platform to redistribute surplus food efficiently and sustainably.

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Firebase
- Google Maps API 
- AI-assisted decision making

## 📊 Expected Impact

- Reduce up to **95%** of non-recoverable food waste
- Support diversion of **730+ metric tons** of food annually
- Potential reduction of **550+ tons of CO₂ emissions**
